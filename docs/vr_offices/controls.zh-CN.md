# 操作说明（VR Offices）

本文记录 `vr_offices/` 目前的键盘/鼠标操作绑定，避免以后忘记。

## 镜头（对话框关闭时）

- 旋转视角：按住 **鼠标右键** 拖动
- 平移视角：按住 **鼠标中键** 拖动
- 缩放：**鼠标滚轮**

## NPC 交互（对话框关闭时）

- 选中 NPC：对 NPC **鼠标左键单击**
- 移动选中 NPC：在地板上 **鼠标右键单击**（包含工作区内部）
- 进入对话：对 NPC **鼠标左键双击**（效果等同于按 **E**）
- 进入对话（键盘）：先选中 NPC，然后按 **E**

## 工作区（对话框关闭时）

- 创建工作区：在地板上 **鼠标左键按住拖拽**，松开后输入名称创建
- 工作区菜单：对工作区 **鼠标右键单击**（若已选中 NPC，用 **Shift + 鼠标右键**）
- 删除工作区：工作区菜单 → **Delete workspace**

## 工作区家具：站立办公桌（Standing Desk）

- 开始摆放：对工作区 **鼠标右键单击** → **Add Standing Desk…**
- 摆放模式（进入后会在屏幕底部显示操作提示）：
  - 移动预览：移动鼠标
  - 确认摆放：**鼠标左键单击**
  - 取消摆放：**Esc**
  - 旋转：**R** 或 **鼠标右键单击**（每次 90°，0°/90°/180°/270° 循环）

## 对话框（对话框打开时）

- 退出对话（键盘）：按 **两下 Esc**
  - 第 1 下：结束输入（释放输入框焦点）
  - 第 2 下：关闭对话框
- 退出对话（鼠标，在对话框面板以外的黑色遮罩层点击）：
  - **鼠标右键单击**，或
  - **鼠标左键双击**

输入说明：

- 对话框打开时，UI 会接管鼠标输入，避免触发 3D 镜头的旋转/缩放/平移。

NPC 行为说明：

- “右键点地板移动”会临时关闭 NPC 的随机游走。
- 执行移动命令期间会用更快的“跑步/冲刺”速度（若模型有 `sprint` 动画也会播放）。
- NPC 到达目标点后会进入 `idle`，并开始大约 60 秒的“等待工作中倒计时”。
- 倒计时结束且没有新的指令时，NPC 会恢复随机游走。
- 右键点击处会出现一个淡黄色光圈指示器，NPC 到达后自动消失。
